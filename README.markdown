# prawn-gmagick

A [graphicsmagick](http:////www.graphicsmagick.org/) binding to add support for
additional image formats to [prawn](https://github.com/prawnpdf/prawn).

## Usage

**Gemfile**

```ruby
gem "prawn-gamgick"
```

**Your code**

```ruby
require "prawn-gmagick"
```
